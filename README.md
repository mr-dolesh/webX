# Campus Lost & Found System

A web-based lost and found management system for university campuses.

## Features

- **Admin Dashboard**: Manage all lost and found reports
- **Student Portal**: Report lost/found items and browse listings
- **Search & Filter**: Find items by category, keyword, and status
- **Status Tracking**: Track items as pending, active, or returned

## Demo Login

- **Admin**: admin@university.edu (any password)
- **Student**: user@university.edu (any password)

## Categories

- Electronics
- Accessories
- Books & Stationery
- Clothing
- IDs & Cards
- Other

## How to Use

1. Open `index.html` in your web browser
2. Select your role (Admin or Student)
3. Login with demo credentials
4. Start reporting or browsing items!

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling (required)
- `script.js` - JavaScript functionality (required)

## Requirements
we need just a normal web browser.
